# 🚀 Guide de Connexion - Nouveau Système Simple

## 🎯 Connexion en 3 Clics

### Étape 1 : Accéder à l'Application
- Ouvrez l'application dans votre navigateur
- L'écran de connexion s'affiche automatiquement

### Étape 2 : Sélectionner votre Profil  
- **Regardez la liste** des utilisateurs disponibles
- **Cliquez sur votre carte** avec votre nom
- Votre sélection sera **mise en évidence en bleu**

### Étape 3 : Se Connecter
- Cliquez sur le bouton **"Se connecter en tant que [Votre nom]"**
- L'application s'ouvre immédiatement
- Vous êtes connecté pour **8 heures**

## 🔄 Actions Disponibles

### ✅ Se Connecter
- **Condition** : Avoir sélectionné un utilisateur
- **Bouton** : "Se connecter en tant que [Nom]"
- **Résultat** : Accès immédiat à l'application

### 🔄 Actualiser la Liste
- **Utilité** : Recharger les utilisateurs depuis la base
- **Quand utiliser** : Si votre nom n'apparaît pas
- **Bouton** : "Actualiser la liste"

### 🚪 Se Déconnecter (une fois connecté)
- **Emplacement** : Menu "Réglages" → "Déconnexion"
- **Confirmation** : "Voulez-vous vraiment vous déconnecter ?"
- **Résultat** : Retour à l'écran de connexion

## 🆘 Que Faire Si...

### 😕 Votre nom n'apparaît pas ?
1. **Cliquez** sur "Actualiser la liste"
2. **Attendez** quelques secondes le rechargement
3. **Contactez** l'administrateur si toujours absent

### 🚫 Impossible de se connecter ?
1. **Vérifiez** qu'un utilisateur est bien sélectionné (coche bleue)
2. **Essayez** avec un autre navigateur
3. **Actualisez** la page (F5)

### ⏰ Session expirée ?
1. **Normal** après 8 heures d'inactivité
2. **Sélectionnez** à nouveau votre profil
3. **Reconnectez-vous** normalement

### 🔄 Page ne charge pas ?
1. **Vérifiez** votre connexion internet
2. **Actualisez** la page (F5)
3. **Videz** le cache du navigateur si nécessaire

## 🎯 Cas d'Usage Typiques

```
1. Ouvrir l'application
2. Cliquer sur "Marie Martin"
3. Cliquer sur "Se connecter en tant que Marie"
```

### 🔄 Changement d'Équipe
```
1. Menu Réglages → Déconnexion
2. Sélectionner le nouveau profil d'équipe  
3. Se reconnecter avec la nouvelle identité
4. Toutes les actions suivantes sont tracées avec la nouvelle identité
```

### 👥 Poste Partagé
```
1. Utilisateur A se déconnecte
2. Utilisateur B sélectionne son profil
3. Se connecte avec son identité
4. Ses actions sont tracées sous son nom
```
